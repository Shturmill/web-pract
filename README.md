# web-pract — ServiceBox

Учебный проект сервисного центра: 
- frontend на HTML/CSS/Vanilla JS 
- backend на FastAPI
- БД SQLite.


**Реализовано:**
- Выбор услуги в калькуляторе показывает календарь цен.
- Чем ближе дата к сегодняшней, тем выше коэффициент цены.
- Заявки сохраняются в SQLite, а не в localStorage.
- У заявки есть жизненный цикл: `open`, `in_progress`, `done`.
- Мастер видит две доски: свободные заявки и свои заявки в работе.
- Валидация есть на frontend и backend: телефон, имя, устройство, комментарий, дата, код мастера, ID мастера.

## Запуск

Перед первым запуском создайте `.env` из примера и задайте код мастера:

```bash
cp .env.example .env
```

Затем:

```bash
docker compose up --build
```

Открыть:

```text
http://localhost:8080
```

Проверка API:

```text
http://localhost:8080/api/health
```
