О проекте

Сервисный центр с frontend, FastAPI backend и SQLite

Проект не копирует чужой бренд: он повторяет типовую структуру сервиса ремонта, но заявки, профили и переписка теперь сохраняются через backend и SQLite.

Что реализовано

  • Связанные HTML-страницы с общей шапкой, навигацией и футером.
  • Семантические теги: header, nav, main, article, footer.
  • Адаптивные сетки на CSS Grid/Flexbox и медиазапросы по ширине.
  • Отдельные правила для портретной и альбомной ориентации экрана.
  • Светлая и тёмная тема с сохранением выбора в localStorage.
  • Профиль клиента и профиль мастера: заявки, очередь, принятие в работу и сообщения через FastAPI + SQLite.
  • Мини-калькулятор стоимости ремонта на чистом JavaScript.
  • Статусы заявки: open, in_progress, done.
  • Защита от SQL-инъекций через параметризованные запросы SQLite.

Стек

HTML5 CSS3 Vanilla JavaScript FastAPI SQLite Docker

Проект запускается как единое FastAPI-приложение: backend отдаёт API и статические HTML/CSS/JS-файлы.
